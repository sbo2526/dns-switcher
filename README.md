## 📦 امکانات
- تغییر DNS با یک کلیک
- پشتیبانی از DNSهای محبوب (Google, Cloudflare, Shecan و ...)
- ذخیره و استفاده از DNSهای سفارشی
- تست Ping DNSها
- طراحی زیبا و مدرن
